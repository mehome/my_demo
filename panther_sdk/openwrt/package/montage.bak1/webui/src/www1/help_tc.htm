DNS
域名系统是用來把域名和IP位址互相轉換的數據庫. ISP至少會提供一組DNS主機的位置.如果你想要使用其他的DNS主機請在這幾個欄位之中的其中一個輸入IP位置. 你最多可以輸入3組DNS主機的位置, 路由器將利用這些資訊更快地存取域名系统主機.
DHCP(動態主機設置協議)
從ISP自動獲取一個IP位址.
MAC位址複製功能
將WAN端的MAC位址改為LAN端客戶的MAC位址.
靜態IP
設定網際網路上路由器的IP位址與子網路遮罩.
預設閘道器
ISP應該告知您閘道器的IP位址.
MTU(最大傳輸單元)
指定網路傳輸所使用的封包大小上限.
持續連線選項
持續連線,甚至當連線閒置.
時區
請根據你的位置選擇時區
端口觸發
不用指定轉發到某個IP位址的一種端口轉發功能. 根據端口觸發規則, 當發現觸發端口上有出境封包, 接著將允許公用服務端口上入境的封包被轉發到觸發的LAN端客戶. 這被叫做端口觸發因為是由出境封包觸發入境封包的端口轉發功能.
LAN設置
設定LAN端IP位址與子網路遮罩
DHCP(動態主機設置協議)
使用DHCP來得到IP網路上的必要訊息.這將減少網路管理者的負擔,並允許裝置加入網路不需要手動設置.
DHCP伺服器探測
偵測LAN端是否有其他DHCP伺服器.如果偵測到其他DHCP伺服器,將關閉路由器的DHCP伺服器.
DHCP使用者列表
可以看到LAN端使用者的IP位址,主機名稱,MAC位址.另外,可針對特定主機給予IP位址.
虛擬區域網路
虛擬局域網路允許你隔離廣播域的範圍。在相同虛擬區網路的成員有相同的識別
PVID
每個端口VID範圍是1 ~ 4094,路由器模式下,WAN端口VID不能等於LAN端口的VID
優先權
802.1p 使用者其權限0為最低, 7為最高
Group Settings
Up to eight VLAN group rules can config and enable to classfied VLAN identifier and port membership. There are three options on each port, select X for any port that is not included in the VLAN group. Select the Tag option to enable tagging for the port. Select the Untag option to include the port in the VLAN group without tagging.
Management VID
在路由器模式時,Wan端口的VID不能和任一LAN端口的VID相同
PVID settings
Each port has it's port VID and use to classfied VLAN membership if untag packet.
虛擬伺服器
允許建立網路上公開的服務,像是網站,FTP伺服器,e-mail伺服器. 利用端口轉發的功能將封包從路由器上的公用服務端口轉發到LAN端的私用服務端口.
端口映射
允許建立網路上公開的服務,像是網站,FTP伺服器,e-mail伺服器,或是一些特殊的網際網路應用. 特殊的網際網路應用指的是視訊會議或是線上遊戲. 當網際網路上有這些服務要求發生的時候,將轉送這些要求到指定的PC上.
路由表
顯示當前所有路由規則
路由規則
路由規則決定路由器收到的封包要轉送往哪一個IP位址
靜態路由
靜態路由是由用戶或網路管理者手動設置的路由規則(例如: 目的網路IP: 192.168.100.1，子網路遮罩: 255.255.255.0，閘道器IP: 192.168.0.2)
UPnP (通用即插即用)
啟動UPnP後,會自動搜尋LAN端支援通用即插即用的裝置.
DDNS(動態DNS)
將定時刷新路由器IP位址給DDNS伺服器
回復設定
系統設定被回復為出廠設定.注意!!!出廠設定:[使用者帳號: admin]，[密碼: admin]，[IP位址: 192.168.0.1]，[網路遮罩: 255.255.255.0]
VPN(虛擬私人網路)

FTP(文件傳輸協議)

網路會議

URL過濾

MAC過濾

使用者過濾
