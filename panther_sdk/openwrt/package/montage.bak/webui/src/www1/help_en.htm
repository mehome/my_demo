DNS
The Domain Name System (DNS) is how the Internet translates domain or website names into Internet addresses or URLs. Your ISP will provide you with at least one DNS Server IP address. If you wish to utilize another, enter that IP address in one of these fields. You can enter up to three DNS Server IP addresses here. The router will utilize these for quicker access to functioning DNS servers.
DHCP
Obtain an IP address automatically from the ISP (Internet Service Provider).
MAC Address Clone
You can use the "Clone MAC Address" button to copy the MAC address of the Ethernet Card installed by your ISP and replace the WAN MAC address with this MAC address.
Static IP Mode
The router's IP Address and Subnet Mask as seen by external users on the Internet (including your ISP). If your Internet connection requires a static IP address, then your ISP will provide you with a Static IP Address and Subnet Mask.
Default Gateway
Your ISP will provide you with the Gateway IP Address.
MTU (Maximum Transmission Unit)
Specify the largest packet size permitted for Internet transmission.

Keep Alive Option
Keeps you connected to the Internet indefinitely, even when your connection sits idle.
Time Zone

Port Triggering
Allow you to do port forwarding without setting a fixed PC. By setting Port Triggering rules, you can allow inbound traffic to arrive at a specific LAN host, using ports different than those used for the outbound traffic. This is called port triggering since the outbound traffic triggers to which ports inbound traffic is directed.
LAN Setting
The router IP Address and Subnet Mask as seen on the internal LAN. The default value is 192.168.0.1 for IP Address and 255.255.255.0 for Subnet Mask.
DHCP (Dynamic Host Configuration Protocol)
DHCP is a protocol used to obtain the information necessary for operation in an Internet Protocol network. This protocol reduces system administration workload, allowing devices to be added to the network with little or no manual intervention.
DHCP Sever Probe
Enable Probe will detect whether other DHCP server on LAN side. When detected other DHCP server, probe will turnoff route's DHCP server, or it will restart DHCP server.
DHCP Client List
Allow you to see which clients are connected to the Router via IP address, host name, and MAC address.You can select static to fixed it.
Virtual LAN
Virtual LAN allow you to separate range of broadcast domain. Members in the same VLAN have the same VLAN identifier.
PVID
Each port VID range is from 1 ~ 4094. WAN port VID can't be equal to anyone of LAN port's VID in router mode.
Priority
802.1p user priority 0 is the lowest priority while 7 is the highest.
Group Settings
Up to eight VLAN group rules can config and enable to classfied VLAN identifier and port membership. There are three options on each port, select X for any port that is not included in the VLAN group. Select the Tag option to enable tagging for the port. Select the Untag option to include the port in the VLAN group without tagging.
Management VID
Management VLAN identifier can be changed to suit the administrator's preference in bridge mode.
PVID settings
Each port has it's port VID and use to classfied VLAN membership if untag packet.
Virtual Server
Allow you to set up public services on your network, such as web servers, ftp servers, e-mail servers, or other specialized Internet applications. Specialized Internet applications are any applications that use Internet access to perform functions such as videoconferencing or online gaming. When users send this type of request to your network via the Internet, the router will forward those requests to the appropriate PC.

Routing Table
The routing table display the current routing information in system.
Static Routing
A static route is a pre-determined pathway that network information must travel to reach a specific host or network. (For example: Destination Network IP: 192.168.100.1, Subnet Mask: 255.255.255.0, Gateway IP: 192.168.0.2)
UPnP (Universal Plug and Play)
UPnP allows automatic discovery and configuration of equipment attached to your LAN. UPnP is supported by Windows ME, XP, or later. It provides compatibility with networking equipment, software and peripherals of the over 400 vendors that cooperate in the Universal Plug and Play forum.
DDNS (Dynamic DNS)
DDNS provides you on the Internet with a method to tie their domain name to a computer or server. DDNS allows your domain name to follow your IP address automatically by having your DNS records changed when your IP address changes.
Reset default
System configuration is reset to the factory default settings. Default settings are: "Username: admin","Password: admin","IP: 192.168.0.1","Netmask: 255.255.255.0

URL Filter
Url Filtering allowing you to prevent users from accessing specified websites on the basis of some policy.
MAC Address Filter
The MAC address filter enables you to allow or restrict specified nodes from communicating with other nodes.
Client Filter
