請參考說明手冊
靜態IP
設定網際網路上路由器的IP位址與子網路遮罩.
預設閘道器
ISP應該告知您閘道器的IP位址.
MTU(最大傳輸單元)
指定網路傳輸所使用的封包大小上限.
時區
請根據你的位置選擇時區
端口觸發
不用指定轉發到某個IP位址的一種端口轉發功能. 根據端口觸發規則, 當發現觸發端口上有出境封包, 接著將允許公用服務端口上入境的封包被轉發到觸發的LAN端客戶. 這被叫做端口觸發因為是由出境封包觸發入境封包的端口轉發功能.
LAN設置
設定LAN端IP位址與子網路遮罩
DHCP(動態主機設置協議)
使用DHCP來得到IP網路上的必要訊息.這將減少網路管理者的負擔,並允許裝置加入網路不需要手動設置.
DHCP伺服器探測
偵測LAN端是否有其他DHCP伺服器.如果偵測到其他DHCP伺服器,將關閉路由器的DHCP伺服器.
DHCP使用者列表
可以看到LAN端使用者的IP位址,主機名稱,MAC位址.另外,可針對特定主機給予IP位址.
端口映射
允許建立網路上公開的服務,像是網站,FTP伺服器,e-mail伺服器,或是一些特殊的網際網路應用. 特殊的網際網路應用指的是視訊會議或是線上遊戲. 當網際網路上有這些服務要求發生的時候,將轉送這些要求到指定的PC上.
路由表
顯示當前所有路由規則
路由規則
路由規則決定路由器收到的封包要轉送往哪一個IP位址
靜態路由
靜態路由是由用戶或網路管理者手動設置的路由規則(例如: 目的網路IP: 192.168.100.1，子網路遮罩: 255.255.255.0，閘道器IP: 192.168.0.2)
回復設定
系統設定被回復為出廠設定.注意!!!出廠設定:[使用者帳號: admin]，[密碼: admin]，[IP位址: 192.168.0.1]，[網路遮罩: 255.255.255.0]
VPN(虛擬私人網路)
根據您的VPN使用需要打開對應的穿透功能
FTP(文件傳輸協議)
讓您LAN端的FTP伺服器可以支援被動模式; 讓LAN端的FTP客戶可以支援主動模式
網路會議
為了讓網路會議可以被發起, 監控端口1720.
URL過濾
設定使用者不能存取的網站
MAC過濾
設定禁止連上網際網路的使用者
使用者過濾
封鎖使用者特定連線的建立，請設定使用者的IP位址 端口編號(80:HTTP) 傳輸協定(TCP/IP)
防火牆選項
啟用網路攻擊防護功能
狀態
此頁面顯示路由器當前的狀態和設定
韌體版本
可上網www.vendor.com查看是否有更新的韌體
DNS
路由器使用的DNS伺服器IP位址，通常會設定多個DNS伺服器，以第一個設定為先
韌體更新
請先確認韌體檔案已經儲存在主機的硬碟內，接著按下Browse選擇韌體
工具
如果系統沒有回應，可按下重啟鍵進行強制重啟
語言
指定網頁顯示的語言
Email紀錄
Email路由器紀錄資訊到收件者電子郵件地址
SMTP伺服器
寄件者的郵件伺服器
電子郵件信箱
請先設定好寄件與收件者的電子郵件地址，再按傳收進行收件測試
登出
登出帳號，需重新輸入帳號密碼!!
